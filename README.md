## 旅游爱好者社交网站

## 前言
毕业整一年, 无意间整理移动硬盘资料的时候翻到了自己的毕业设计, 看了看原有的代码(JSP+Servlet+JDBC+Oracle), 大量重复代码, 一脸的嫌弃。于是痛定思痛, 打算重新开发。 一来是为了练练手, 让自己更好的去掌握和使用新的技术; 二来也算是给自己一个交代, 毕竟毕设也没有认真对待。<br />

## 环境要求
- JDK1.8

- JQuery（[查看JQuery API中文文档](http://www.jquery123.com/)）
